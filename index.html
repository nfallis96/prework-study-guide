HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • HTML provides the physical content of a web application.

CSS

  • CSS stands for cascading styling sheets.
  • CSS allows to change fonts, colors, sizes, and other design elements of your web application.
  • CSS selectors are used to target individual elements within your HTML.
  • A period before a class name in CSS indicates styling for any element with that class.

Git

  • git status: checks what branch we are currently on and what files are being tracked.
  • git checkout -b "branch name": Creates a new branch and switches to it.
  • It is very important to pull content from the main branch after completing a merge.

JavaScript

  • JavaScript allows you to add functionalities to your web application.
  • JavaScript uses functions to provide instructions for different page actions.
